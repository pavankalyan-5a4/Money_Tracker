# 💰 Money Tracker (Core Java)

A simple console-based expense tracker built using Core Java.

## 📌 Features

* Automatically uses current date
* Add expense with amount and short note
* View all expenses
* Show total spending

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts
* ArrayList

## ▶️ How to Run

1. Compile the code:
   javac *.java

2. Run the program:
   java Main

## 📸 Sample Output

2026-04-14 | ₹50.0 | Tea
